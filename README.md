# React Native on HarmonyOS

A simple weather app built with React Native to showcase how the framework runs on **HarmonyOS** compared to iOS and Android.

## Getting Started

```sh
npm install
npm start
```

```sh
npm run ios        # iOS
npm run android    # Android
npm run harmony    # HarmonyOS
```

### Prerequisites

- **iOS / Android**: [React Native Environment Setup](https://reactnative.dev/docs/set-up-your-environment)
- **HarmonyOS**: [RNOH Getting Started](https://gitcode.com/openharmony-sig/ohos_react_native/blob/0.77.1-rc.1-ohos/docs/en/getting-started.md)
